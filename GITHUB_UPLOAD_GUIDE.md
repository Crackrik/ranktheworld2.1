## The Super Easy Way For Complete Beginners

### Step 1: Download These Files

You should have these files:
- `index.html` (the main page)
- `app.js` (the code that makes it work)
- `data.csv` (100,000 items to rank)
- `README.md` (description)
- `SETUP.md` (instructions)
- `LICENSE` (legal stuff)
- `.gitignore` (tells git what to ignore)

OR just use `rankeverything-repo.tar.gz` (all files in one zip)

---

### Step 2: Create GitHub Account

1. Go to **github.com**
2. Click **"Sign up"** (top right)
3. Enter email, password, username
4. Verify your email
5. Done!

---

### Step 3: Create New Repository

1. After logging in, click the **"+"** button (top right corner)
2. Click **"New repository"**

3. Fill in:
   - **Repository name**: `rankeverything` (or whatever you want)
   - **Description**: "Rank everything using ELO scores"
   - **Public** (select this - it's free!)
   - âœ… Check "Add a README file" (UNCHECK THIS - we have our own)
   - Click **"Create repository"**

---

### Step 4: Upload Your Files

You're now on your new empty repository page.

**Option A: Drag and Drop (Easiest)**

1. Click **"uploading an existing file"** (blue text in the middle of page)
2. Drag ALL the files into the box
   - OR click "choose your files" and select them all
3. Wait for upload to finish
4. Scroll down and click **"Commit changes"** (green button)

**Option B: Use GitHub Desktop (If You Want)**

1. Download GitHub Desktop from desktop.github.com
2. Install it
3. Click "Clone Repository" 
4. Select your `rankeverything` repo
5. Copy all your files into that folder
6. In GitHub Desktop, click "Commit to main"
7. Click "Push origin"

---

### Step 5: Enable GitHub Pages

1. In your repository, click **"Settings"** (tab at top)
2. Scroll down the left sidebar and click **"Pages"**
3. Under **"Source"**:
   - Branch: Select **"main"** (or "master")
   - Folder: Select **"/ (root)"**
4. Click **"Save"**
5. Wait 2-3 minutes (GitHub is building your site)

---

### Step 6: Visit Your Site!

After 2-3 minutes, your site will be live at:

```
https://YOUR-USERNAME.github.io/rankeverything
```

Replace `YOUR-USERNAME` with your actual GitHub username.

For example, if your username is `coolperson123`, go to:
```
https://coolperson123.github.io/rankeverything
```

## Troubleshooting

**"My site shows a 404 error"**
- Wait 5 minutes (GitHub needs time to build it)
- Check Settings â†’ Pages to see if it's enabled
- Make sure files are in the root folder (not in a subfolder)

**"It's blank/broken"**
- Make sure you uploaded ALL the files
- Check that `data.csv` is there
- Open browser console (F12) to see errors

**"I can't find the Settings tab"**
- You need to be logged in
- You need to be on YOUR repository page
- Settings is at the top right (after Code, Issues, Pull Requests)
